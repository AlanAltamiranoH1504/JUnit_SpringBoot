# Investigación de JUnit y Mockito

## Descripción

Este repositorio contiene una investigación sobre **JUnit** y **Mockito**, dos de las herramientas más utilizadas en el desarrollo de aplicaciones Java para la realización de pruebas unitarias y simulación de dependencias.

El objetivo principal es comprender el funcionamiento, características y ventajas de estas herramientas, así como aplicar sus conceptos mediante ejemplos prácticos.

---

## Objetivos

- Conocer los fundamentos de las pruebas unitarias en Java.
- Comprender la estructura y funcionamiento de JUnit.
- Analizar el uso de Mockito para la creación de objetos simulados (mocks).
- Aplicar pruebas unitarias utilizando JUnit y Mockito.
- Identificar las ventajas de implementar pruebas automatizadas en proyectos de software.

---

## Tecnologías Utilizadas

- Java
- JUnit 5
- Mockito
- Maven
- IDE (IntelliJ IDEA)
